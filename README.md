## shellの設定
```
// Mac
$ curl -L raw.github.com/snoozers/dotfiles/master/etc/init/setup.sh | bash

// Debian
$ apt update && apt-get install -y curl && curl -L raw.github.com/snoozers/dotfiles/master/etc/init/setup.sh | bash
```

## アプリケーションのインストール
```
$ make install
```

## その他
```
$ make
```
